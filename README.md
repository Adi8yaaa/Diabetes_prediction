# Diabetes Prediction App using SVM

A Machine Learning web application that predicts whether a person is likely to have diabetes based on medical parameters.  
The model is built using **Support Vector Machine (SVM)** and deployed with **Streamlit**.

---

## 📌 Features

- Predicts diabetes likelihood instantly
- User-friendly web interface using Streamlit
- Machine Learning model trained with SVM
- Fast and lightweight deployment
- Input validation for user data

---

## 🛠️ Technologies Used

- Python
- Scikit-learn
- Pandas
- NumPy
- Streamlit
- Pickle
